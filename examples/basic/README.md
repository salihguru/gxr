# GXR Basic Example

This is a basic example of using the GXR framework.

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Build client components:
```bash
npm run build
# or
npx gxr build
```

3. Run the Go server:
```bash
go run .
```

4. Open http://localhost:8080

## Project Structure

```
basic/
├── client/
│   ├── components/
│   │   ├── Counter.tsx    # "use client" - interactive component
│   │   └── Header.tsx     # Server component - static
│   └── pages/
│       └── index.tsx      # Page template
├── public/
│   └── hydrate.js         # Generated by `gxr build`
├── main.go                # Go server
└── package.json
```

## How It Works

1. `Counter.tsx` has `"use client"` directive at the top
2. `npx gxr build` scans for these components and generates `hydrate.js`
3. Go renders the full page with SSR
4. The browser loads `hydrate.js` which makes Counter interactive
